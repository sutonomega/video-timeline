# Video Timeline

A tool for extracting events from videos and turning them into readable timelines.

## Goal

Turn video scenes into structured timeline logs that can be used for review, commentary, or AI context.

## Features

- Scene detection
- Timeline generation
- Event extraction
- JSON export
